This directory is meant for the integrated PDF textbooks.

To integrate a textbook, simply upload the PDF file into this folder matching the strict naming convention:

- Class 6 Mathematics: `class_6_mathematics.pdf`
- Class 6 Computer Science: `class_6_computer_science.pdf`
- Class 7 Mathematics: `class_7_mathematics.pdf`
- Class 8 Computer Science: `class_8_computer_science.pdf`
(etc.)

Upload these files using the AI studio file explorer into this `/public/textbooks/` folder. The app will automatically detect them and include them in the LLM context!
